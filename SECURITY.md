# Aviso de Seguridad y Uso Ético

## Propósito del Proyecto

Este proyecto fue creado con fines recreativos y educativos únicamente. Su objetivo es demostrar:
- La automatización de correos electrónicos usando Node.js
- El uso de variables de entorno para la seguridad
- La implementación de servicios SMTP
- Prácticas básicas de programación asíncrona

## ⚠️ Advertencia de Uso

Este script NO debe ser utilizado para:
- Spam o envío masivo de correos no deseados
- Acoso o molestia a terceros
- Saturación de servidores de correo
- Cualquier actividad maliciosa o no ética

## Uso Responsable

Para usar este proyecto de manera responsable:
1. Obtén consentimiento explícito del destinatario
2. Usa frecuencias de envío razonables
3. Respeta los límites de las APIs de Gmail
4. No compartas credenciales ni información sensible

## Consecuencias Legales

El mal uso de este script puede resultar en:
- Violación de términos de servicio de Gmail
- Suspensión de cuentas de correo
- Posibles acciones legales por acoso
- Violación de leyes anti-spam

## Descargo de Responsabilidad

Los creadores y contribuyentes de este proyecto:
- No se hacen responsables del mal uso del código
- No apoyan ni promueven actividades de acoso
- Recomiendan usar el código solo con fines educativos y recreativos
- Se reservan el derecho de no dar soporte a usos no éticos

## Reporte de Problemas

Si descubres un uso indebido de este código o tienes preocupaciones de seguridad, por favor:
1. No utilices el código para fines maliciosos
2. Reporta el uso indebido a las autoridades correspondientes
3. Contacta con los administradores del repositorio

Recuerda: La tecnología debe usarse para crear, no para molestar.
